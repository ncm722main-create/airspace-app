Vertical Speed Tape
drawHudVSbar() / drawProfile() — ±2500 fpm
// Visibility gate
show IF fwdNm ≤ warnWhiteNm
    AND timeToEntry ≤ warnWhiteSec

// Danger band calculation
vsToFloor = (as.floor - alt) / minToEntry
vsToCeil  = (as.ceil  - alt) / minToEntry
band = [min(floor,ceil) … max(floor,ceil)]
1Probe forward along heading for closest airspace entry distance
2Compute time to entry from speed; skip if outside white thresholds
3Calculate fpm range that would put aircraft inside airspace
4Red if current VS is inside band or within 200 fpm; yellow if within 500 fpm
+2.5K fpm ← danger band: VS that enters Class C 0 fpm —— -2.5K fpm
Corridor Bar (Lateral)
drawCorridorBar() — bottom center, left/right proximity
// Only nearest airspace per side
show IF edgeDist ≤ warnWhiteLatNm

// Bar width proportional to distance
barWidth = (dist / warnWhiteLatNm) × halfWidth
// Closer = wider fill toward center
1For each airspace at current altitude, compute closest edge on each side
2Keep only the single nearest airspace per side (left / right)
3Bar width scales linearly: closer airspace = wider fill toward center

Alert Thresholds (configurable in Settings)

Red Alert — Flashes 4 Hz
Forward: ≤ 1.0 nm or ≤ 30 sec
Vertical: ≤ 500 ft
Lateral: ≤ 0.5 nm
Immediate threat — flashing red overlay on class color
Yellow Alert — Flashes 2 Hz
Forward: ≤ 2.0 nm or ≤ 60 sec
Vertical: ≤ 1000 ft
Lateral: ≤ 1.0 nm
Caution — flashing yellow overlay on class color
White (Visibility Gate)
Forward: ≤ 5.0 nm or ≤ 300 sec
Lateral: ≤ 3.0 nm
Controls bar visibility only — shown in airspace class color, no flash
